About Me

By day, I'm a software developer focused on oil & gas accounting software and government financial systems.

By night, I advocate for game developers in Alberta.

  • 2012: Moved to Calgary
  • 2013: Founded the Calgary Game Developers meetup group.
  • 2015: Co-founded the Calgary Game Developers Association NPO. Became its first president.
  • 2018: Founded Alberta Makes Games (a game development advocacy project)
  • 2019: Joined Interactive Arts Alberta as a board member.
  • 2016-2020: Global Game Jam site organizer (Calgary).
  • 2021: Global Game Jam regional organizer (Canada).
  • 2021: Transferred the Alberta Makes Games website to Digital Alberta.
